# KWENDA Brand Dashboard

Desktop web dashboard for tracking sales, social media, planning, action plans, and future drops.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Recharts
- lucide-react (icons)

All data in `src/data/sampleData.js` is fake sample data — replace with real API/database calls when ready.
